• append <string> – adds the specified one word string at the end of the list and prints the list (space separated).
  • prepend <string> – inserts the specified one word string at the start of the list and prints the list.
  • reverse – reorders the list items in reversed order (from its end to its start) and prints the list.
  • insert <index> <string> – inserts the specified one word string at the specified position in the list and prints the list after that.
  • delete <index> – deletes the one word string at the specified position in the list and prints the list.
  • roll left – rolls the list on the left and prints the list. The first list item comes last.
  • roll right – rolls the list on the right and prints the list. The last list item comes first.
  • sort – sorts the list in alphabetical order and prints the list.
  • count <string> – prints how many times the specified one word string occurs in the list (case sensitive).
  • end – stops the execution of commands. Print “Finished” as command output.
